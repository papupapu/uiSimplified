# uiSimplified

Defining the way to create templates and components for React powered websites.

# Installation

Once you installed node and yarn, you should be able to set up the environment by runnin 'yarn' from the command line.
Then just type 'yarn run dev' to start webpack development server, or 'yarn run build' to create a production build.

Finally, type 'node server.js' in the root directory and open 'http://localhost:8080/' in a browser window.
